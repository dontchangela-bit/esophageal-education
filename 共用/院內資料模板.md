# 院內資料填寫說明

> 本文件提供統一的院內資料填寫格式。所有衛教文件中標記為「院內資料區」的區塊，請參考以下範本自行填入。

---

## 填寫格式說明

在每份衛教 Markdown 文件中，您會看到以下標記：

```markdown
<!-- 🏥 院內資料區 - 請自行填入 -->
> **📋 請填入貴院資料：**
>
> - 本院負責科別：_______________
> - 聯絡電話 / 分機：_______________
> - 門診時間：_______________
> - 主治醫師：_______________
> - 本院特色 / 年手術量：_______________
<!-- 院內資料區結束 -->
```

請直接編輯 `_______________` 處，填入貴院實際資訊。

---

## 範例（填寫完成後）

```markdown
<!-- 🏥 院內資料區 -->
> **📋 本院資料：**
>
> - 本院負責科別：胸腔外科 / 肝膽腸胃科
> - 聯絡電話 / 分機：(02) 2312-3456 分機 6789
> - 門診時間：週一、三、五上午 / 週二下午
> - 主治醫師：王大明 主任、李小華 醫師
> - 本院特色 / 年手術量：年均 POEM 手術 50 例，微創食道切除 30 例
<!-- 院內資料區結束 -->
```

## 各主題可能需要填入的院內資訊

### 食道弛緩不能症
- 負責科別（肝膽腸胃科 / 胸腔外科）
- POEM 手術經驗與年手術量
- Heller Myotomy 手術經驗
- 氣球擴張治療能量
- 門診掛號方式與時間
- 住院天數（院內平均）
- 術後追蹤流程

### 食道功能檢查
- 檢查室位置（樓層 / 棟別）
- 使用之 HRM 設備品牌與型號
- pH-Impedance 監測設備
- 報告產出時間
- 預約方式（門診開單 / 自行預約）
- 檢查費用（健保 / 自費項目）
- 特殊注意事項（本院專屬流程）

### 食道癌微創手術
- 手術團隊組成（外科、腫瘤科、放射科等）
- 年手術量與 MIE/RAMIE 比例
- 達文西機器手臂可用性
- 術前評估流程（本院多專科團隊會議）
- 平均住院天數
- 術後加護病房照護
- 營養諮詢與復健資源
- 癌症個案管理師聯繫方式
