# 食道功能檢查前準備

## 前言

充分的檢查前準備是確保檢查順利進行、結果準確可靠的重要關鍵。本篇將詳細說明各項食道功能檢查前的準備事項，請仔細閱讀並遵循醫護人員的指示。

---

## 一、禁食要求

### 所有食道功能檢查的共同要求

- **檢查前至少禁食 6 小時**（含固體食物和液體）
- 少量清水可於檢查前 2 小時前飲用，之後請完全禁食禁水
- 禁食的目的：
  - 避免檢查過程中嘔吐造成危險（吸入性肺炎，aspiration pneumonia）
  - 確保食道內是空的，不影響檢查結果
  - 減少檢查中的不適感

### 禁食時間速查表

| 項目 | 最後可進食 / 飲用時間 |
|------|---------------------|
| 固體食物（正餐、點心） | 檢查前 **6 小時以上** |
| 牛奶、含渣飲品 | 檢查前 **6 小時以上** |
| 清澈液體（清水、無渣果汁） | 檢查前 **2 小時以上** |
| 口香糖、糖果 | 檢查前 **6 小時以上** |

> **實用建議**：如果您的檢查安排在早上，最簡單的做法是**前一天晚上 12 點後不吃不喝**。

---

## 二、藥物調整（非常重要）

部分藥物會影響食道功能檢查的結果，特別是在做**逆流監測 (reflux monitoring)** 時。請務必與您的醫師確認是否需要停藥。

### 需要停用的藥物

#### 質子幫浦抑制劑 (Proton Pump Inhibitor, PPI)

停藥時間：**檢查前 7 天**

常見藥物名稱：
- 耐適恩 (Nexium / esomeprazole)
- 百抑潰 (Pariet / rabeprazole)
- 泰克胃通 (Takepron / lansoprazole)
- 歐克胃 (Omeprazole)
- 泮托拉唑 (Pantoprazole)

#### H2 受體拮抗劑 (H2 Receptor Antagonist)

停藥時間：**檢查前 3 天**

常見藥物名稱：
- 乏乃迪 (Famotidine)
- 乃迪 (Ranitidine)（部分地區已下架）
- 西米替丁 (Cimetidine)

#### 制酸劑 (Antacid)

停藥時間：**檢查前 6 至 12 小時**

常見藥物名稱：
- 胃乳片 / 胃乳液
- 鋁鎂混合劑
- 氧化鎂 (MgO)
- 碳酸鈣 (Calcium carbonate) 嚼錠

#### 腸胃蠕動促進劑 (Prokinetics)

停藥時間：**檢查前 3 天**（請遵醫囑）

常見藥物名稱：
- 莫沙比利 (Mosapride)
- 多潘立酮 (Domperidone)

### 停藥時間總覽

| 藥物類別 | 停藥時間 | 常見藥名舉例 |
|---------|---------|------------|
| PPI（質子幫浦抑制劑） | 檢查前 7 天 | Nexium、Pariet、Takepron |
| H2 拮抗劑 | 檢查前 3 天 | Famotidine |
| 制酸劑 | 檢查前 6-12 小時 | 胃乳片、胃散 |
| 腸胃蠕動促進劑 | 檢查前 3 天 | Mosapride |

### 重要注意事項

> **請注意**：以上停藥建議主要針對**逆流監測檢查**（pH 監測、pH-阻抗監測、Bravo 監測）。如果您只做**食道壓力測定 (HRM)**，停藥要求可能不同，請依醫師指示為準。

> **請勿自行停藥**：停藥前請務必與您的主治醫師確認。某些藥物（如血壓藥、心臟藥、抗癲癇藥等）不可隨意停用。

## 三、特殊狀況的準備

### 糖尿病 (Diabetes) 患者

- 檢查當天早上**暫停**口服降血糖藥物 (oral hypoglycemic agents)
- 使用胰島素 (insulin) 的患者，請提前與醫師討論劑量調整
- 檢查前請自行測量血糖，若血糖過低請告知醫護人員
- 建議將檢查安排在**早上第一個時段**，以減少禁食時間
- 隨身攜帶糖果或含糖飲料，以備檢查後使用

### 服用抗凝血劑 (Anticoagulant) 或抗血小板藥物 (Antiplatelet) 的患者

- 阿斯匹靈 (Aspirin)：通常**不需要停藥**
- 華法林 (Warfarin)、新型口服抗凝血劑 (NOACs)：請與醫師討論是否需要調整
- 氯吡格雷 (Clopidogrel)：請與醫師討論

### 孕婦或可能懷孕者

- 請務必在檢查前告知醫護人員
- 食道功能檢查本身不涉及輻射，基本上是安全的
- 但醫師需要評估檢查的必要性和時機

### 鼻腔或喉嚨問題

- 有嚴重鼻塞、鼻息肉 (nasal polyp)、鼻中隔彎曲 (deviated septum) 的患者，請事先告知
- 過去做過鼻腔或喉嚨手術者，請事先告知
- 醫師可能需要調整導管放置的方式

## 四、檢查當天該帶什麼？

### 必帶物品清單

- [ ] 健保卡 (NHI card)
- [ ] 身分證或其他身分證件
- [ ] 目前使用的**藥物清單**或藥袋
- [ ] 醫師開立的檢查單或轉介單
- [ ] 過去相關的檢查報告（如有）

### 建議攜帶物品

- [ ] 舒適寬鬆的衣物（特別是 24 小時監測需配戴記錄器）
- [ ] 有前扣或拉鍊的上衣（方便穿脫和裝置記錄器）
- [ ] 少量現金或信用卡（部分檢查可能有自費項目）
- [ ] 手機和充電器（24 小時監測期間可正常使用手機）
- [ ] 紙筆或手機記事功能（記錄症狀和用餐時間）
- [ ] 輕食或點心（檢查完成後食用）

### 穿著建議

- **建議穿著**：舒適的棉質上衣，最好是有拉鍊或釦子的開襟款式
- **避免穿著**：緊身衣物、高領上衣、連身洋裝
- **24 小時監測額外建議**：腰帶或有口袋的褲子（方便攜帶記錄器）

## 五、交通安排

### 需要有人陪同的情況

| 檢查類型 | 是否需要人陪同接送 |
|---------|-----------------|
| HRM（壓力測定） | 通常不需要，可自行離開 |
| 24h pH / pH-阻抗監測 | 通常不需要（放置和取下時各來一次） |
| Bravo 無線監測 | **需要**（因可能使用鎮靜劑） |
| FLIP | **需要**（通常在胃鏡鎮靜下進行） |

> **重要提醒**：如果檢查過程中使用了鎮靜劑 (sedative) 或麻醉 (anesthesia)，**請勿自行開車或騎車**。請事先安排家人或朋友接送，或使用大眾運輸 / 計程車。

## 六、心理準備

### 常見擔憂與建議

**「我很怕管子從鼻子插入」**
- 醫護人員會先在鼻腔噴上麻醉藥，大幅降低不適感
- 管子很細（約 4 毫米），比胃鏡細很多
- 放置過程通常只需 1-2 分鐘
- 配合醫護人員的指令做吞嚥動作，會更加順利

**「我擔心會一直想吐」**
- 初始放置時可能有輕微嘔吐反射 (gag reflex)，這是正常的
- 導管就位後，嘔吐感通常會逐漸消退
- 深呼吸和放鬆有助於減輕不適
- 如果不適感持續，請告知醫護人員

**「24 小時帶著管子生活會很不方便嗎？」**
- 大部分日常活動都可以正常進行
- 可以正常工作（輕度辦公工作）
- 可以正常說話和進食
- 建議避免激烈運動和淋浴（保護記錄器）

## 檢查前準備完整清單

以下是您可以列印使用的準備清單：

### 檢查前 7 天
- [ ] 依醫囑停用 PPI（如有需要）
- [ ] 確認回診及取下記錄器的時間（如適用）

### 檢查前 3 天
- [ ] 依醫囑停用 H2 拮抗劑（如有需要）
- [ ] 依醫囑停用腸胃蠕動促進劑（如有需要）

### 檢查前 1 天
- [ ] 確認檢查時間和地點
- [ ] 安排交通方式（如需鎮靜）
- [ ] 準備好需要攜帶的物品
- [ ] 選好合適的衣物

### 檢查前 6-12 小時
- [ ] 依醫囑停用制酸劑
- [ ] 開始禁食（固體食物和牛奶等）

### 檢查當天
- [ ] 檢查前 2 小時起完全禁食禁水
- [ ] 攜帶所有必要物品
- [ ] 穿著舒適寬鬆衣物
- [ ] 糖尿病患者：測量血糖，依醫囑調整藥物
- [ ] 準時到達檢查地點
