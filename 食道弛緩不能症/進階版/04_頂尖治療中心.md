# 食道弛緩不能症（Esophageal Achalasia）— 頂尖治療中心

## 前言

食道弛緩不能症的治療效果與醫療團隊的經驗密切相關，尤其是 POEM（經口內視鏡肌肉切開術）和 Heller 手術等技術。**高手術量中心（High-Volume Centers）**的治療成功率和安全性通常顯著優於低手術量中心。

SAGES 2024 指引明確**強烈建議**弛緩不能症的治療應在有足夠經驗的中心執行。

本篇列出美國與台灣在食道弛緩不能症領域具有代表性的治療中心，供臨床參考。

## 選擇治療中心的考量因素

### 對患者的建議

在選擇治療中心時，建議考量以下因素：

| 考量項目 | 說明 |
|----------|------|
| **手術量** | 高手術量中心的成功率和安全性通常較佳。建議選擇年 POEM 手術量 ≥ 10~20 例的中心 |
| **術者經驗** | 詢問主刀醫師的個人經驗和累積案例數 |
| **完整團隊** | 理想的中心應有消化內科、消化外科、食道功能檢測實驗室的完整配置 |
| **追蹤系統** | 是否有完善的術後追蹤計畫（包括 pH 監測、內視鏡追蹤等） |
| **處理併發症的能力** | 是否有能力處理手術相關併發症（如穿孔等） |
| **地理便利性** | 長期追蹤需要定期回診，距離是實際考量 |
| **二線治療能力** | 若初次治療失敗，是否有多種再治療選項 |

### 對醫療專業人員的建議

| 轉介指標 | 建議 |
|----------|------|
| 診斷困難的病例 | 轉介至有完整食道功能檢測的中心 |
| Type III 弛緩不能症 | 建議轉介至有 POEM 經驗的中心 |
| 先前治療失敗 | 轉介至高手術量中心評估二線選項 |
| 乙狀食道 / 巨食道症 | 轉介至有處理複雜食道疾病經驗的中心 |
| 年輕患者 | 轉介至可提供多種治療選項並有長期追蹤系統的中心 |

## 品質指標參考

| 品質指標 | 建議目標 |
|----------|----------|
| POEM 術後 Eckardt Score ≤ 3 的比率 | ≥ 85% |
| POEM 相關嚴重併發症率 | < 3% |
| LHM 食道穿孔率 | < 5% |
| PD 穿孔率 | < 3% |
| 術後 30 天再入院率 | < 5% |
| 術後 GERD 監測完成率（POEM 後） | ≥ 80% |
| 術後 1 年追蹤完成率 | ≥ 80% |

## 重點整理

| 重點 | 說明 |
|------|------|
| 治療中心選擇很重要 | 高手術量中心的成功率和安全性顯著較佳 |
| 美國代表性中心 | Stanford（POEM 高量）、NYU（POEM 先驅）、Mayo（診斷頂尖）、U of Chicago（分類發源地） |
| 台灣代表性中心 | 台大醫院（POEM 先驅）、台中慈濟（90% 成功率）、亞東、台北榮總、馬偕 |
| 選擇考量 | 手術量、術者經驗、完整團隊、追蹤系統、地理便利性 |
| 轉介時機 | Type III、治療失敗、乙狀食道等複雜病例建議轉介 |
