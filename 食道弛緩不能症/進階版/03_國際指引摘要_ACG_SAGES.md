# 食道弛緩不能症（Esophageal Achalasia）— 國際指引摘要

## 概述

本篇彙整目前與食道弛緩不能症診療最相關的國際臨床指引，包括：

1. **ACG Clinical Guidelines**（美國胃腸病學會臨床指引）
2. **ASGE 2020 Guideline**（美國消化內視鏡學會 2020 年指引）
3. **SAGES 2024 POEM Update**（美國消化內視鏡外科醫師學會 2024 年 POEM 更新）
4. **EAES 2026 Multi-Society Guideline**（歐洲內視鏡外科學會 2026 年多學會指引，制定中）

所有指引均採用 **GRADE 方法學**（Grading of Recommendations, Assessment, Development and Evaluations）進行證據評估與建議分級。

---

## GRADE 方法學簡介

| 證據品質 | 定義 |
|----------|------|
| **高（High）** | 進一步研究不太可能改變對療效估計的信心 |
| **中等（Moderate）** | 進一步研究可能對療效估計的信心有重要影響，且可能改變估計 |
| **低（Low）** | 進一步研究很可能對療效估計的信心有重要影響，且很可能改變估計 |
| **極低（Very Low）** | 對療效估計非常不確定 |

| 建議強度 | 定義 |
|----------|------|
| **強烈建議（Strong）** | 多數患者適用，臨床醫師應遵循 |
| **有條件建議（Conditional）** | 視個別患者情況而定，需考量患者偏好與臨床判斷 |

## 一、ACG Clinical Guidelines

### 指引背景
- 發布機構：美國胃腸病學會（American College of Gastroenterology, ACG）
- 採用 GRADE 方法學
- 涵蓋食道弛緩不能症的診斷與治療完整建議
- 首次正式採納芝加哥分類（Chicago Classification）的三個亞型系統

### 診斷相關建議

| 建議內容 | 證據品質 | 建議強度 |
|----------|----------|----------|
| 建議使用高解析度食道測壓（HRM）作為診斷食道弛緩不能症的首選工具 | 低 | 有條件建議 |
| 所有疑似弛緩不能症患者應接受上消化道內視鏡以排除假性弛緩不能症 | 極低 | 強烈建議 |
| 鋇劑吞嚥攝影（Barium Esophagram）可作為輔助診斷工具，評估食道型態與排空 | 極低 | 有條件建議 |
| 應依據 HRM 結果將弛緩不能症分為 Type I、II、III 三個亞型 | 低 | 強烈建議 |
| 定時鋇劑排空測試（Timed Barium Esophagram, TBE）可用於治療後追蹤 | 極低 | 有條件建議 |

### 治療相關建議

| 建議內容 | 證據品質 | 建議強度 |
|----------|----------|----------|
| 對適合治療的弛緩不能症患者，建議接受確定性治療（PD、LHM 或 POEM）而非僅藥物或 BTX | 中等 | 強烈建議 |
| PD（氣球擴張術）和 LHM（Heller 肌肉切開術）均為有效的一線治療 | 中等 | 強烈建議 |
| PD 建議採用分級擴張方案（Graded Dilation Protocol），從 30mm 氣球開始 | 低 | 有條件建議 |
| LHM 建議合併部分胃底摺疊術以降低術後 GERD | 中等 | 強烈建議 |
| BTX 注射建議保留給不適合接受 PD 或手術的患者 | 中等 | 強烈建議 |
| 藥物治療（鈣離子通道阻斷劑、硝酸鹽）僅建議作為其他治療的過渡 | 低 | 有條件建議 |
| 治療後應定期追蹤評估症狀改善與併發症 | 極低 | 有條件建議 |

### ACG 關於亞型與治療選擇

| 亞型 | ACG 治療建議 |
|------|-------------|
| Type I | PD 或 LHM 均可，證據未明確顯示哪個較佳 |
| Type II | PD 或 LHM 均可，此亞型對各種治療反應最佳 |
| Type III | 建議 LHM 或 POEM，PD 效果較差 |

## 二、ASGE 2020 Guideline

### 指引背景
- 發布機構：美國消化內視鏡學會（American Society for Gastrointestinal Endoscopy, ASGE）
- 發布年份：2020
- 聚焦於內視鏡在弛緩不能症中的診斷與治療角色

### 關鍵建議摘要

| 建議內容 | 證據品質 | 建議強度 |
|----------|----------|----------|
| POEM 是弛緩不能症的有效治療選項，短期療效與 LHM 相當 | 中等 | 有條件建議 |
| POEM 術後應監測 GERD，考慮常規 pH 監測和內視鏡追蹤 | 低 | 有條件建議 |
| 分級氣球擴張（Graded PD）仍為弛緩不能症的有效一線治療 | 中等 | 強烈建議 |
| 內視鏡在弛緩不能症的角色包括診斷（排除假性弛緩不能症）和治療（BTX、PD、POEM） | — | 專家共識 |
| BTX 注射適用於高手術風險或暫時性治療的患者 | 中等 | 有條件建議 |

### ASGE 對 POEM 的技術建議

| 技術面向 | ASGE 建議 |
|----------|----------|
| 隧道入口位置 | 距 EGJ 約 10~12 cm |
| 切開方向 | 前壁（Anterior）或後壁（Posterior）均可 |
| 肌肉切開範圍 | 食道端至少 6 cm，胃端延伸 2~3 cm 超過 EGJ |
| Type III 的特殊考量 | 可延長食道端切開至 > 10 cm 以涵蓋痙攣段 |
| 隧道關閉 | 使用內視鏡夾（Endoscopic Clips）關閉黏膜入口 |

## 三、SAGES 2024 POEM Update

### 指引背景
- 發布機構：美國消化內視鏡外科醫師學會（Society of American Gastrointestinal and Endoscopic Surgeons, SAGES）
- 發布年份：2024
- 這是 SAGES 首次針對 POEM 發布的獨立更新指引
- 納入了最新的隨機對照試驗（Randomized Controlled Trials, RCTs）和系統性回顧

### 核心建議

| 建議內容 | 證據品質 | 建議強度 |
|----------|----------|----------|
| **POEM 可作為弛緩不能症的一線確定性治療** | 中等 | 有條件建議 |
| **建議 POEM 優先於 PD（氣球擴張術）** | 低至中等 | 有條件建議 |
| **Type III 弛緩不能症建議首選 POEM** | 低 | 有條件建議 |
| POEM 與 LHM 的短中期療效相當，可作為替代選項 | 中等 | 有條件建議 |
| POEM 術後應常規評估 GERD 風險，考慮 PPI 預防或治療 | 低至中等 | 有條件建議 |
| POEM 應在有足夠經驗的中心執行 | 專家共識 | **強烈建議** |
| 術者應完成足夠的訓練（建議 ≥ 20 例有指導的操作） | 專家共識 | 強烈建議 |

### SAGES 2024 的重要更新觀點

**1. POEM 優先於 PD 的依據：**
- 多項 RCT 顯示 POEM 的 2 年症狀緩解率優於 PD
- POEM 的再治療率低於 PD
- POEM 提供更持久的括約肌減壓效果

**2. POEM vs LHM 的定位：**
- 兩者短中期療效相當（2~5 年數據）
- LHM + Fundoplication 在 GERD 控制方面有優勢
- 選擇應基於：亞型、GERD 風險、術者經驗、患者偏好
- SAGES 未明確建議 POEM 優先於 LHM（兩者並列為一線選項）

**3. GERD 管理建議：**
- 所有 POEM 術後患者應被告知 GERD 風險
- 建議術後 3~12 個月內進行客觀 GERD 評估（pH 監測 +/- 內視鏡）
- 有 GERD 症狀者應給予 PPI
- 長期追蹤食道黏膜變化

## 指引間的比較與整合

### 各指引對主要治療的定位比較

| 治療方式 | ACG | ASGE 2020 | SAGES 2024 |
|----------|-----|-----------|------------|
| POEM | 提及為治療選項（發布較早） | 有效治療選項，與 LHM 相當 | **一線治療，優先於 PD** |
| LHM | 一線治療 | 一線治療 | 一線治療（與 POEM 並列） |
| PD | 一線治療 | 有效一線治療 | 有效但建議 POEM 優先 |
| BTX | 保留給特殊患者 | 高風險/暫時性治療 | 未特別強調 |

### 各指引的核心差異

| 比較項目 | ACG | ASGE 2020 | SAGES 2024 |
|----------|-----|-----------|------------|
| POEM 的地位 | 較保守（早期數據） | 等同 LHM 的替代 | 明確的一線推薦 |
| Type III 建議 | LHM 或 POEM | 未特別區分 | **POEM 首選** |
| POEM vs PD | 未直接比較 | 未明確優先順序 | **POEM 優先於 PD** |
| GERD 管理 | 一般性建議 | 建議監測 | 具體追蹤建議 |

## 臨床應用建議

### 綜合各指引的實務建議

1. **診斷流程**：遵循 ACG 建議的標準化診斷流程（EGD → HRM → 分型 → TBE）
2. **治療選擇**：
   - Type I / II：POEM、LHM、PD 均為合理選項，依患者因素和醫師專長決定
   - Type III：優先考慮 POEM（SAGES 2024）
   - 不適合手術者：BTX 注射
3. **POEM 執行標準**：應在有經驗的中心進行（SAGES 2024 強烈建議）
4. **術後追蹤**：POEM 後應常規監測 GERD；所有治療後應以 Eckardt Score 追蹤
5. **再治療**：初次失敗後有多種選項，依個案評估

## 重點整理

| 重點 | 說明 |
|------|------|
| 最新趨勢 | POEM 的地位持續上升，從替代選項到一線推薦 |
| SAGES 2024 關鍵訊息 | POEM 優先於 PD；Type III 首選 POEM |
| 各指引共識 | LHM 和 POEM 短中期療效相當；BTX 僅適用特殊患者 |
| 待解決問題 | POEM 長期 GERD 影響、> 10 年療效數據 |
| 即將發布 | EAES 2026 多學會指引，預期提供更全面建議 |
