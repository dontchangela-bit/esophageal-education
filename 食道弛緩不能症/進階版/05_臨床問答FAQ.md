# 食道弛緩不能症（Esophageal Achalasia）— 臨床問答 FAQ

## 關於亞型與治療選擇

### Q1：哪個亞型對 POEM 的治療反應最好？

**Type II（泛食道加壓型）的治療反應最佳**，對所有治療方式（POEM、LHM、PD）的成功率都是三型中最高的，POEM 成功率可達 90~95%。

原因在於 Type II 患者的食道壁仍保有一定程度的肌肉功能（體現為泛食道加壓現象）。當 LES 阻力透過 POEM 被移除後，食道腔內的壓力可以有效推動食物通過。

**Type III（痙攣型）**雖然對傳統治療（PD、LHM）反應較差，但對 POEM 的反應卻相當好（85~95%），因為 POEM 可向食道近端延長肌肉切開至 > 10 cm，涵蓋痙攣區段。這是 SAGES 2024 建議 Type III 首選 POEM 的主要依據。

**Type I（經典型）**通常代表疾病進展較久，食道已明顯擴張，治療反應居中（POEM 成功率約 80~90%）。

---

### Q2：什麼情況下應選擇氣球擴張（PD）而非肌肉切開術（POEM/LHM）？

考慮 PD 而非 POEM/LHM 的情境包括：

| 情境 | 理由 |
|------|------|
| 患者強烈希望避免全身麻醉 | PD 可在鎮靜下完成 |
| 醫療資源有限，無法提供 POEM/LHM | PD 技術門檻相對較低 |
| 孕婦需要暫時緩解 | 全麻手術風險的考量 |
| 等待確定性治療前的過渡 | 提供症狀緩解 |
| 患者身體狀況不適合長時間手術 | PD 操作時間短 |

然而，SAGES 2024 已有條件地建議 POEM 優先於 PD，主要基於 POEM 的中期療效較持久且再治療率較低。若患者適合且有條件接受 POEM，一般建議優先考慮。

**PD 的分級擴張方案（Graded Dilation Protocol）：**
- 第一次使用 30 mm 氣球
- 若效果不佳，4~6 週後可用 35 mm 氣球重複
- 必要時可進一步用 40 mm 氣球
- 每次擴張前需排除穿孔

### Q3：POEM 術後 GERD 如何管理？

POEM 術後 GERD 的發生率（20~50%）是目前臨床上最受關注的問題之一。管理策略如下：

**術後常規評估：**
- 術後 3~12 個月建議進行 24 小時 pH 監測（或 pH-Impedance 監測）
- 術後 1 年建議內視鏡評估食道黏膜（Los Angeles Classification 分級）
- 即使無症狀，也建議客觀評估（約 15~30% 的患者有無症狀性酸暴露升高）

**藥物管理：**
- 有症狀者或客觀檢查證實異常酸暴露者：開始 PPI（Proton Pump Inhibitor）治療
- 常用藥物：Omeprazole 20mg、Esomeprazole 40mg、Pantoprazole 40mg 等，每日一次
- 多數患者以標準劑量 PPI 即可有效控制
- 部分患者可能需要長期使用

**難治性 GERD 的處理：**
- PPI 加倍劑量
- 加入 H2 受體拮抗劑（H2RA）睡前服用
- 評估是否有其他原因（如食道排空不良造成的症狀）
- 極少數難以控制者可考慮抗逆流手術（如 Fundoplication），但需權衡再次手術風險

**長期監測：**
- 每 1~3 年內視鏡追蹤，監測 Barrett's 食道（Barrett's Esophagus）的發展
- POEM 術後 > 10 年的 Barrett's 食道發生率數據仍在累積中

### Q4：初次治療失敗後有哪些選項？

治療失敗的定義通常為 Eckardt Score > 3 或客觀檢查（TBE、HRM）未見改善。

**失敗後的系統性評估：**
1. 確認診斷正確（排除假性弛緩不能症）
2. 評估初次治療是否充分（切開長度、擴張程度）
3. 評估食道型態變化（是否進展為乙狀食道）
4. 考慮是否有纖維化或瘢痕形成

**二線治療選項：**

| 初次治療 | 建議再治療選項 | 成功率 | 備註 |
|----------|--------------|--------|------|
| PD 失敗 | POEM 或 LHM | 70~85% | 兩者皆有良好證據 |
| POEM 失敗 | 重複 POEM（不同方向）、PD、LHM | 60~80% | 後壁 POEM 失敗可改前壁，或反之 |
| LHM 失敗 | POEM（後壁入路）或 PD | 60~75% | POEM 可避開前壁手術區域 |
| 多次失敗 | 食道切除術（Esophagectomy） | — | 最後手段；手術風險較高 |

**POEM 作為救援治療（Redo POEM）的特殊優勢：**
- 可選擇與先前不同的進入方向（前壁 vs 後壁）
- 黏膜下隧道的方式可避開先前手術的瘢痕區域
- 但技術難度較高，建議在高手術量中心執行

## 關於診斷與評估

### Q5：FLIP（Functional Lumen Imaging Probe）在弛緩不能症中的角色為何？

FLIP 是一種利用阻抗面積測量（Impedance Planimetry）原理的食道功能評估工具，可測量：

- **EGJ 擴張性指數（EGJ Distensibility Index, EGJ-DI）**：弛緩不能症患者的 EGJ-DI 顯著降低
- **食道體部的收縮模式（FLIP Panometry）**：可識別有無蠕動和收縮模式

**臨床應用：**
| 應用情境 | 角色 |
|----------|------|
| HRM 結果不確定時 | 輔助診斷，提供額外功能資訊 |
| EGJOO 的鑑別 | 幫助區分真正的弛緩不能症早期和功能性 EGJOO |
| 術中評估（Intraoperative FLIP） | POEM 或 LHM 中即時評估肌肉切開是否充分 |
| 術後追蹤 | 客觀評估 EGJ 通透性改善程度 |
| 研究用途 | 弛緩不能症病理生理的進一步了解 |

**目前定位：**
- FLIP 尚未被列為常規診斷工具，但在特殊情境中的價值逐漸被認可
- 術中 FLIP 的使用正在增加，有研究顯示可改善手術成功率
- EAES 2026 指引預計將對 FLIP 的角色提供更具體的建議

### Q6：食道測壓（Manometry）追蹤的時機與意義？

**治療後 HRM 追蹤的意義：**
- 客觀評估 LES 壓力是否降低
- 評估 IRP 的變化（治療成功的客觀指標之一）
- 但需注意：術後 HRM 的解讀與術前不同，已切開的括約肌不一定會顯示「正常」的測壓圖

**建議追蹤時機：**

| 時間點 | HRM 追蹤建議 |
|--------|-------------|
| 術後 3 個月內 | 通常不建議（食道仍在恢復） |
| 術後 3~6 個月 | 若症狀未改善，可考慮 HRM 評估 |
| 術後 1 年 | 建議進行 HRM 作為客觀療效評估 |
| 症狀復發時 | 應進行 HRM 重新評估 |

**HRM 追蹤的注意事項：**
- 術後 IRP 降至正常範圍是治療成功的良好指標
- 但部分治療成功（症狀改善）的患者，IRP 可能仍略高於正常上限
- 應結合症狀（Eckardt Score）和客觀檢查（TBE、HRM）綜合判斷

### Q7：乙狀食道（Sigmoid Esophagus）如何處理？

乙狀食道是弛緩不能症長期進展的結果，食道嚴重擴張並彎曲成 S 型。

**分級（依嚴重度）：**

| 分級 | 食道直徑 | 食道型態 |
|------|---------|---------|
| 輕度擴張 | < 4 cm | 直線或輕微彎曲 |
| 中度擴張 | 4~6 cm | 可能有輕度彎曲 |
| 重度擴張（巨食道症） | > 6 cm | 明顯 S 型彎曲 |

**治療策略：**

| 嚴重度 | 治療建議 | 預期效果 |
|--------|---------|---------|
| 輕至中度 | POEM 或 LHM 仍可嘗試 | 成功率略低於非擴張者，但多數仍有效 |
| 重度但功能尚存 | POEM/LHM + 密切追蹤 | 可能需要多次介入 |
| 重度且功能喪失 | 考慮食道切除術（Esophagectomy） | 手術風險較高，但可根本解決問題 |

**乙狀食道 POEM/LHM 的特殊挑戰：**
- 內視鏡或腹腔鏡操作技術難度增加
- 食道走向異常可能影響隧道建立（POEM）
- 即使括約肌壓力降低，擴張的食道排空功能仍可能不佳
- 需在高手術量中心由經驗豐富的術者執行

## 關於特殊族群

### Q8：POEM 在高齡患者中的安全性與效果？

**整體而言，POEM 在高齡患者（> 65~70 歲）中仍是安全且有效的治療。**

| 比較項目 | 高齡組（> 65 歲） | 非高齡組 |
|----------|-----------------|---------|
| 症狀緩解率 | 80~90% | 85~95% |
| 嚴重併發症率 | 略高（3~5%） | 1~3% |
| 住院天數 | 可能略長（2~4 天） | 1~3 天 |
| 術後 GERD | 發生率類似 | 類似 |
| 整體安全性 | 可接受 | 良好 |

**高齡患者的特殊考量：**
- 術前心肺功能評估更為重要
- 全身麻醉的風險需仔細評估
- 若全麻風險過高，可考慮 PD（鎮靜下進行）或 BTX 注射
- 治療目標可能更著重於**症狀緩解和營養維持**，而非追求完美的功能恢復

---

### Q9：弛緩不能症患者使用的藥物是否會影響食道功能？

**需要注意的藥物：**

| 藥物類別 | 對食道功能的影響 | 建議 |
|----------|-----------------|------|
| 鴉片類止痛藥（Opioids） | 可降低 LES 放鬆，加重吞嚥困難 | 盡量避免或減量 |
| 抗膽鹼藥物（Anticholinergics） | 抑制食道蠕動 | 注意可能的影響 |
| 鈣離子通道阻斷劑（CCB） | 放鬆 LES，可暫時緩解症狀 | 可作為過渡性治療 |
| 硝酸鹽類（Nitrates） | 放鬆 LES | 可作為過渡性治療 |
| GLP-1 受體促效劑 | 可能延遲胃排空 | 術前需考慮停藥時間 |
| PPI | 不影響食道運動功能 | 安全使用，POEM 術後可能需要 |

**術前用藥注意：**
- 建議在治療前將所有用藥告知醫師
- 某些藥物可能影響 HRM 結果，需在檢查前停藥（依醫師指示）
- GLP-1 受體促效劑（如 Semaglutide）因延遲胃排空，全麻前可能需提前停藥

---

### Q10：弛緩不能症與食道癌的長期風險關聯？

**風險數據：**
- 弛緩不能症患者的食道鱗狀細胞癌（Squamous Cell Carcinoma, SCC）風險為一般人群的 10~50 倍（但基礎發生率極低）
- 估計長期（20~30 年）累積風險約 1~3%
- 風險可能與食道長期食物滯留、慢性發炎有關
- POEM 術後的 GERD 是否增加腺癌（Adenocarcinoma）風險，尚待長期數據

**追蹤建議：**
- 治療後建議每 1~3 年進行上消化道內視鏡追蹤
- 重點觀察食道黏膜的變化（發炎、化生、異形成）
- POEM 術後同時監測 GERD 相關黏膜變化（Barrett's 食道）
- 若發現異形成（Dysplasia），需更密集的追蹤和處理

## 關於治療技術細節

### Q11：POEM 前壁（Anterior）vs 後壁（Posterior）入路的選擇？

| 比較項目 | 前壁入路（Anterior） | 後壁入路（Posterior） |
|----------|---------------------|---------------------|
| 歷史 | 最早採用的方式 | 後來發展的替代路徑 |
| 解剖特點 | 沿食道前壁建立隧道 | 沿食道後壁（大彎側）建立隧道 |
| 胃端延伸 | 在胃賁門前壁延伸 | 在胃底方向延伸 |
| 術後 GERD | 部分研究顯示可能略高 | 部分研究顯示可能略低（但爭議中） |
| LHM 術後救援 | 可能受限（LHM 為前壁切開） | **較佳選擇**（避開前壁瘢痕） |
| 技術難度 | 兩者相當 | 兩者相當 |

**目前共識：**
- 前壁和後壁入路的療效和安全性大致相當
- 首次 POEM 時兩者皆可
- 先前接受過 LHM 的患者，建議使用後壁入路
- 部分術者基於個人經驗和偏好選擇特定入路

### Q12：POEM 術中併發症的即時處理？

| 術中併發症 | 發生率 | 即時處理 |
|-----------|--------|---------|
| 黏膜穿孔（Mucosal Perforation） | 2~5% | 內視鏡夾（Clips）修補；多不影響預後 |
| 氣腹（Capnoperitoneum） | 5~15% | 腹腔穿刺減壓（Needle Decompression） |
| 氣胸（Pneumothorax） | 1~3% | 胸腔穿刺或胸管引流 |
| 皮下氣腫（Subcutaneous Emphysema） | 5~10% | 通常自行吸收；嚴重者需減壓 |
| 出血 | 1~2% | 內視鏡止血（電凝、止血夾） |
| 隧道迷失方向 | < 1% | 重新定位或放棄手術 |

**術中氣體相關併發症的預防：**
- 使用 CO2 而非空氣進行充氣（CO2 吸收較快）
- 術中持續監測腹腔壓力
- 麻醉團隊監測呼吸參數變化
- 備妥穿刺減壓設備

### Q13：如何評估 POEM 切開長度是否充分？

**術中評估方法：**

| 方法 | 描述 | 優缺點 |
|------|------|--------|
| 直接內視鏡觀察 | 觀察肌肉纖維是否完全切開 | 主觀性較強 |
| 術中 FLIP | 測量 EGJ-DI 是否達到目標值（通常 > 6 mm^2/mmHg） | 客觀、即時回饋，但需額外設備 |
| 術後 TBE | 鋇劑排空改善程度 | 術後才能評估 |
| 術後 HRM | IRP 降低程度 | 需等術後恢復才能進行 |

**切開長度建議：**
- 食道端：至少 6 cm（Type III 可延長至 10~15 cm 以涵蓋痙攣段）
- 胃端：延伸 2~3 cm 超過 EGJ
- 術中 FLIP 正逐漸成為評估切開充分性的重要輔助工具

---

### Q14：弛緩不能症患者的麻醉特殊考量？

| 考量項目 | 說明 | 建議 |
|----------|------|------|
| 誤吸風險（Aspiration Risk） | 食道內食物滯留增加誤吸風險 | 術前至少禁食 12~24 小時；考慮術前內視鏡清除殘留食物 |
| 快速序列誘導（Rapid Sequence Induction, RSI） | 降低誘導期間的誤吸風險 | 麻醉團隊應考慮 RSI |
| 術中通氣 | POEM 中 CO2 充氣可能影響膈肌運動 | 密切監測呼吸參數；備妥減壓設備 |
| 體位 | POEM 通常取仰臥位 | 注意壓瘡預防 |
| 術後止吐 | 預防術後噁心嘔吐（可能增加食道壓力） | 多模式止吐預防 |

### Q15：弛緩不能症的未來治療方向？

| 研究方向 | 目前狀態 | 潛在影響 |
|----------|---------|---------|
| 內視鏡抗逆流介入（Anti-reflux Mucosectomy, ARMS/POEM+F） | 早期臨床研究 | 可能解決 POEM 術後 GERD 問題 |
| 機器人輔助 POEM | 實驗階段 | 可能提高精準度 |
| FLIP 導引的治療決策 | 逐步進入臨床 | 術中即時回饋優化手術 |
| 幹細胞治療（Stem Cell Therapy） | 早期研究 | 理論上可修復退化的神經元，但距臨床應用尚遠 |
| 基因治療 | 基礎研究 | 針對病因的根本治療，非常早期階段 |
| 生物標記物（Biomarkers） | 研究中 | 早期診斷、預測治療反應 |
| AI 輔助診斷 | 發展中 | 自動化測壓圖判讀、亞型分類 |

## 重點整理

| 重點 | 說明 |
|------|------|
| Type II 治療反應最佳 | 對所有治療方式皆然；Type III 首選 POEM |
| POEM 術後 GERD 管理 | 常規 pH 監測 + 內視鏡；多數 PPI 可控制 |
| 治療失敗的處理 | 多種二線選項；POEM 可作為 LHM 後的救援治療 |
| FLIP 的角色日增 | 輔助診斷、術中評估、術後追蹤 |
| 高齡患者 | POEM 仍安全有效，需更仔細的術前評估 |
| 乙狀食道 | 增加治療難度，嚴重者可能需食道切除術 |
| 長期追蹤 | 定期內視鏡監測食道癌風險和 GERD 相關變化 |
